Sistema de Toast - Case Bem

Teste dos Toasts

Clique nos botões abaixo para testar os diferentes tipos de toast:

Como Usar

No JavaScript:

// Métodos diretos
showSuccess('Mensagem de sucesso');
showError('Mensagem de erro');
showWarning('Mensagem de aviso');
showInfo('Mensagem informativa');
showAlert('Mensagem de alerta');

// Método genérico
showToast('Mensagem', 'tipo', duracao);

// Usando o manager diretamente
window.toastManager.success('Sucesso');
window.toastManager.error('Erro');
window.toastManager.warning('Aviso');
window.toastManager.info('Info');
window.toastManager.alert('Alerta');

No Backend (Python):

# Usando funções flash em português
from util.flash_messages import informar_sucesso, informar_erro, informar_aviso

informar_sucesso(request, "Operação realizada!")
informar_erro(request, "Algo deu errado!")
informar_aviso(request, "Cuidado!")
return RedirectResponse("/destino", status_code=303)

# Ou usando variáveis tradicionais
return templates.TemplateResponse("template.html", {
    "request": request,
    "sucesso": "Operação realizada!",
    "erro": "Algo deu errado!",
    "warning": "Cuidado!",
    "info": "Informação importante",
    "alert": "Alerta crítico"
})
